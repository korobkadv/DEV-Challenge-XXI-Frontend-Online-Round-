Запуск:
    1. Скопіювати файли.
    2. Запустити "Live Server" або щось схоже.
    3. Відкрити index.html
    4. Готово, користуйтеся скриптом.

Що вміє:
    1. Працює з файлами json та csv.
    2. Підтримує як завантаження через кнопку так і Drag-n-drop.
    3. Малює "Лінійний графік", "Стовпчикову діаграмму" та "Кругову діаграмму".
    4. Налаштування графіків:
        а) Підпис осі X та Y
        б) Підпис всього графіка
        в) Зміна кольорів графіків та діаграм
    5. Єкспорт в PNG, SVG, PDF
    6. Друк графіка
